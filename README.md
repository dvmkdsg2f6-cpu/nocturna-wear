# NOCTURNA WEAR

🤘 **Rock. Metal. Alternative. Clothing.**

A premium e-commerce platform for alternative and metal clothing enthusiasts.

## Features

- ✅ Modern Next.js 13+ with App Router
- ✅ Dark theme optimized for metal/rock aesthetic
- ✅ Fully responsive design
- ✅ Interactive product catalog
- ✅ Ready for Vercel deployment

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
git clone https://github.com/dvmkdsg2f6-cpu/nocturna-wear.git
cd nocturna-wear
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
npm run start
```

## Deployment on Vercel

1. Go to [Vercel](https://vercel.com)
2. Click "Import Project"
3. Select your GitHub repository: `nocturna-wear`
4. Click "Deploy"

Vercel will automatically detect Next.js and optimize your deployment.

## Customization

### Change Brand Colors
Edit `app/globals.css` and modify the color values:
- Primary background: `#050505`
- Accent color: `#ffff00` (yellow)

### Add Products
Edit the `products` array in `app/page.js`

### Add Pages
Create new files in the `app/` directory (e.g., `app/about/page.js`)
